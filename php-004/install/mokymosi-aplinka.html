Susikurti PHPStorm projektą

Susikurti naują PHP projektą klonuojant jį (checkout) iš GitHUB (per VCS). Tam reikės su copy/paste nurodyti GitHUB projekto URL.

Susieti projektą su Web serveriu

  1. Pasileisti WAMP (turi degti žalia spalva apačioje, jeigu viskas ok).
  2. Interneto naršyklėje prisijungti prie WAMP valdymo puslapio įvedus URL'ą: localhost
  3. Paspausti "Add a virtual host"
  4. Laukelyje "Name of the Virtual Host" įvesti testinės svetainės adresą, pvz. test.lt
  5. PHPStorm projekte viršuje (projekto medyje) paspausti dešinį pelės klavišą ir pasirinkti "Copy path"
  6. WAMP valdymo puslapio laukelyje "Complete absolute path of the VirtualHost folder" atlkti paste (perkelti PHPStorm projekto path)
  7. Sukurti virtual host (mygtukas "Start the creation of the Virtual Host")
  8. Apačioje ant žalios WAMP ikonos paspausti pelės dešinį klavišą, pasirinkti tools->restart DNS

Pratestuoti mokymosi aplinką

PHPStorm projekte viršuje (projekto medyje) paspausti dešinį pelės klavišą ir pasirinkti new->html file. Įvesti kokį nors tekstą body sekcijoje. Interneto naršyklėje įvesti virtual host URL'ą (pvz. test.lt) ir pasirinkti sukurtą hTML failą. Turite matyti įvestą tekstą.

PHPStorm projekte viršuje (projekto medyje) paspausti dešinį pelės klavišą ir pasirinkti git->commit diretory - atliekamas pakeitimų išsaugojimas. PHPStorm projekte viršuje (projekto medyje) paspausti dešinį pelės klavišą ir pasirinkti git->repositoty->push - atliekamas projekto failų siuntimas į GitHUB. Pirmą kartą bus prašoma atlikti autorizaciją
